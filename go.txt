Installing Go

Visit https://golang.org/dl/
Find the latest release for Linux ARM64 (ARMv8)

$ wget https://dl.google.com/go/go1.14.2.linux-arm64.tar.gz
$ tar -C /usr/local -xzf go1.14.2.linux-arm64.tar.gz

For fish:

$ echo 'set PATH $PATH /usr/local/go/bin' >> ~/.config/fish/config.fish

For bash:

$ echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.profile
